# 🎵 Audio Organizer Web App  
### Organizador Inteligente de Áudios (com renomeação + STT + remoção de duplicados)

Este projeto transforma seu script Python de organização de áudios em um **aplicativo web real**, com:

- 🌙 **Tema escuro aurora (ciano + violeta suave)**
- 📂 Upload de múltiplos arquivos de áudio
- 🎧 Remoção automática de duplicados
- 🗣 Renomeação inteligente usando Google Speech-to-Text (até 5 palavras)
- 🔠 Prefixos configuráveis
- 📦 Download final como arquivo `.zip`
- 🖥 Funciona em qualquer dispositivo (PC, Android, iPhone)
- ☁ Pode ser hospedado na nuvem (Render.com)
